# Skill: Token Efficiency

## Goal

Guide AI agents to minimize token usage without reducing implementation quality.

## Workflow

1. Define minimal target scope (files/modules).
2. Gather only essential context.
3. Implement directly; avoid speculative explorations.
4. Validate with focused checks during edits.
5. Run full required checks before completion.

## Communication Pattern

- Keep status updates short and informative.
- Do not restate unchanged context.
- Summarize deltas only.

## Anti-Patterns

- Broad repository scans without scope.
- Re-reading unchanged files multiple times.
- Verbose logs and repeated explanations.
