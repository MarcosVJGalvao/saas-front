# Boletim Escolar

## Objetivo

Gerenciar períodos, matriz curricular, lançamentos, consultas, dashboard e processamentos de boletim.

## Rotas

- `/client/report-cards/academic-periods`
- `/client/report-cards/grade-subjects`
- `/client/report-cards/entries`
- `/client/report-cards/processings`

## Critérios de aceite

- Lançamentos respeitam período fechado.
- Enums de boletim traduzidos.
- Consultas usam detalhes e tabelas compartilhadas.
